O next.js tem 3 principais pastas da nossa aplicação:

* public
-> Aqui vão ficar os seus arquivos estáticos(imagens, vídeos, gifs)
* pages
-> fica a nossa página principal renderizada
* styles
-> fica nossos arquivos .css com os estilos tanto globais, quanto o escopado(modularizado), porém não é obrigatório ficar em styles

esse escopado vamos a um exemplo para melhor entendimento, onde foi importado o import styles from '../styles/Home.module.css', toda vez
que é colocado o styles, segue exemplo:

<div className={styles.container}>
      <Head>
        <title>Create Next App</title>
        <meta name="description" content="Generated by create next app" />
        <link rel="icon" href="/favicon.ico" />
      </Head>

onde é colocado styles vai ser aplicado esse estilo.
